# ProofStamp

Decentralized document verification and timestamping.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/ProofStamp.sol`
- Stacks: `contracts/stacks/contracts/proofstamp.clar`

## Setup
```bash
pnpm install
pnpm dev
```
